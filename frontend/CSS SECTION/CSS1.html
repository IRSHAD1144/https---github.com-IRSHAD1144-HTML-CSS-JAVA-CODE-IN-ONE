🎬 ANIMATION PROPERTIES

animation (shorthand)

All animation properties in one

.element {
    animation: myAnimation 2s ease-in-out 0.5s infinite alternate;
}

Animating box (slide effect)

animation-name | animation-duration | animation-timing-function

Animation Name, Duration & Timing

@keyframes slide {
    0% { transform: translateX(0); }
    100% { transform: translateX(100px); }
}
.element {
    animation-name: slide;
    animation-duration: 2s;
    animation-timing-function: ease-in-out;
}

2-second smooth slide animation

animation-delay | animation-iteration-count | animation-direction

Delay, Repeat Count & Direction

.element {
    animation-delay: 2s;
    animation-iteration-count: 3;
    animation-direction: alternate;
}

Plays 3 times, alternates direction

animation-fill-mode | animation-play-state

Fill Mode & Play State

.element {
    animation-fill-mode: forwards; /* keeps end state */
    animation-play-state: paused; /* or running */
}

Forwards keeps final position | Play-state controls running/paused
