🔗 LINK & NAVIGATION ELEMENTS (4)

<a>

Anchor / Hyperlink

<a href="https://www.google.com" target="_blank">Visit Google</a>
         <a href="#top">Go to Top</a>

OUTPUT:

Visit Google
Go to Top

<nav>

Navigation (grouping container)

<nav>
    <a href="#home">Home</a> |
    <a href="#about">About</a> |
    <a href="#contact">Contact</a>
</nav>

OUTPUT:

Home | About | Contact

<link>

External Resource Link

<!-- Used inside <head> -->
<link rel="stylesheet" href="style.css">
<link rel="icon" href="favicon.ico">
<link rel="preconnect" href="https://fonts.com">

OUTPUT:

⚡ Note: <link> elements go in the <head> and don’t display visible content.
They link to stylesheets, icons, fonts, etc.
