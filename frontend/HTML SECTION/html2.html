🎵 MULTIMEDIA ELEMENTS (10)

<img>

Image Tag

<img src="image.jpg" alt="Description">

Output: Displays an image on the page.

<picture>

Picture Tag

<picture>
    <source media="(min-inline-size:650px)" srcset="img_large.jpg">
    <source media="(min-inline-size:465px)" srcset="img_medium.jpg">
    <img src="img_small.jpg" alt="Responsive image">
</picture>

Output: Provides multiple image sources for responsive design.

<source>

Source Tag

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
</audio>

Output: Specifies multiple media resources for audio/video.

<audio>

Audio Tag

<audio controls>
    <source src="song.mp3" type="audio/mpeg">
    Your browser does not support audio.
</audio>
<video>

Video Tag

<video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
    Your browser does not support video.
</video>
<iframe>

Inline Frame

<iframe src="https://www.example.com" width="300" height="200"></iframe>
<embed>

Embed Tag

<embed src="file.pdf" width="300" height="200">

Output: Embeds external content like PDFs, Flash, etc.

<object>

Object Tag

<object data="file.pdf" type="application/pdf" width="300" height="200">
    <p>Alternative text</p>
</object>

Output: Embeds objects like PDFs, images, or applets.

<param>

Parameter Tag

<object data="audio.wav">
    <param name="autoplay" value="true">
    <param name="loop" value="false">
</object>

Output: Defines parameters for plugins embedded with <object>.

<track>

Track Tag

<video controls>
    <source src="video.mp4" type="video/mp4">
    <track src="subtitles_en.vtt" kind="subtitles" srclang="en" label="English">
</video>

Output: Adds text tracks (subtitles/captions) for video/audio.

🏷️ SEMANTIC ELEMENTS (12)

<header>

Header Tag

<header>
    <h1>Website Title</h1>
    <p>Tagline or description</p>
</header>

Website Title

Tagline or description

Output: Represents introductory content or a group of navigational links.

<footer>

Footer Tag

<footer>
    <p>© 2024 Company Name</p>
    <p>Contact: info@example.com</p>
</footer>

© 2024 Company Name

Contact: info@example.com

Output: Represents footer content for a section or page.

<main>

Main Tag

<main>
    <h1>Main Content</h1>
    <p>This is the primary content of the page.</p>
</main>

Main Content

This is the primary content of the page.

Output: Specifies the main content of the document.

<section>

Section Tag

<section>
    <h2>About Us</h2>
    <p>We are a company that does amazing things.</p>
</section>

About Us

We are a company that does amazing things.

Output: Defines a section in a document (like chapters, headers, footers).

<article>

Article Tag

<article>
    <h2>Blog Post Title</h2>
    <p>This is an independent piece of content.</p>
</article>

Blog Post Title

This is an independent piece of content.

Output: Represents independent, self-contained content.

<aside>

Aside Tag

<aside>
    <h4>Related Content</h4>
    <p>This is sidebar information.</p>
</aside>

Related Content

This is sidebar information.

Output: Represents content indirectly related to the main content.

<nav>

Nav Tag

<nav>
    <a href="#">Home</a> |
    <a href="#">About</a> |
    <a href="#">Contact</a>
</nav>
Home About Contact

Output: Defines a set of navigation links.

<figure>

Figure Tag

<figure>
    <img src="image.jpg" alt="Description">
    <figcaption>Image caption here</figcaption>
</figure>

Output: Groups self-contained content like images with captions.

<figcaption>

Figcaption Tag

<figure>
    <img src="diagram.jpg" alt="Diagram">
    <figcaption>Figure 1: System Architecture Diagram</figcaption>
</figure>

Output: Provides a caption for the <figure> element.

<details>

Details Tag

<details>
    <summary>Click to expand</summary>
    <p>This content is hidden until you click the summary.</p>
</details>
Click to expand

This content is hidden until you click the summary.

Output: Creates a disclosure widget that can be opened/closed.

<summary>

Summary Tag

<details>
    <summary>More Information</summary>
    <p>Additional details go here.</p>
</details>
More Information

Additional details go here.

Output: Defines a visible heading for <details> element.

<mark>

Mark Tag (Semantic)

<p>This is a <mark>highlighted</mark> word.</p>

This is a highlighted word.

Output: Highlights text for reference or emphasis.

💻 OTHER ELEMENTS (15)

<script>

Script Tag

<script>
    console.log("Hello World!");
</script>

Open console to see: "Hello World!"

Output: Embeds JavaScript code in the webpage.

<noscript>

NoScript Tag

<noscript>
    This website requires JavaScript to function properly.
</noscript>

Output: Displays content only if JavaScript is disabled in the browser.

<canvas>

Canvas Tag

<canvas id="myCanvas" width="200" height="100" style="border:1px solid #000;"></canvas>
<script>
    var canvas = document.getElementById("myCanvas");
    var ctx = canvas.getContext("2d");
    ctx.fillStyle = "#f87171";
    ctx.fillRect(20, 20, 150, 60);
</script>

Output: Draws graphics on the fly using JavaScript.

<svg>

SVG Tag

<svg width="100" height="100">
    <circle cx="50" cy="50" r="40" stroke="#f87171" stroke-width="4" fill="#34d399" />
</svg>

Output: Draws scalable vector graphics (SVG).

<math>

MathML Tag

<math xmlns="http://www.w3.org/1998/Math/MathML">
    <msup>
        <mi>x</mi>
        <mn>2</mn>
    </msup>
    <mo>+</mo>
    <msup>
        <mi>y</mi>
        <mn>2</mn>
    </msup>
    <mo>=</mo>
    <msup>
        <mi>z</mi>
        <mn>2</mn>
    </msup>
</math>

Output: Displays mathematical formulas (MathML support varies).

<template>

Template Tag

<template id="myTemplate">
    <p>This content is hidden until cloned!</p>
</template>

Output: Stores HTML fragments that can be used later with JavaScript.

<slot>

Slot Tag

<template id="cardTemplate">
    <div>
        <slot name="title">Default Title</slot>
        <slot name="content">Default Content</slot>
    </div>
</template>

Output: Used with shadow DOM for creating web components.

<dialog>

Dialog Tag

<dialog id="myDialog">
    <p>This is a popup dialog box!</p>
    <button onclick="document.getElementById('myDialog').close()">Close</button>
</dialog>
<button onclick="document.getElementById('myDialog').showModal()">Open Dialog</button>

This is a popup dialog box!

Output: Creates a popup dialog box.

<menu>

Menu Tag

<menu>
    <li><button>Copy</button></li>
    <li><button>Paste</button></li>
    <li><button>Cut</button></li>
</menu>
    •

    Output: Defines a list/menu of commands.

    <wbr>

    WBR Tag (Word Break Opportunity)

    <p>ThisIsAReallyLongWordThat<wbr>NeedsToBreak<wbr>AtSpecificPoints</p>

    ThisIsAReallyLongWordThatNeedsToBreakAtSpecificPoints

    Output: Suggests where to break a long word (resize window to see effect).

    <data>

    Data Tag

    <data value="100">One Hundred</data>
    One Hundred

    Output: Associates machine-readable value with content.

    <time>

    Time Tag

    <time datetime="2024-12-25">Christmas 2024</time>
    Christmas 2024

    Output: Represents a specific date/time in machine-readable format.

    <ruby>

    Ruby Tag

    <ruby>
    漢 <rt>かん</rt>
    字 <rt>じ</rt>
</ruby>
    漢 かん 字 じ

    Output: Shows pronunciation annotations (used for East Asian typography).

    <rt>

    RT Tag (Ruby Text)

    <ruby>
    Tokyo <rt>とうきょう</rt>
</ruby>
    Tokyo とうきょう

    Output: Provides pronunciation info for ruby annotations.

    <rp>

    RP Tag (Ruby Parentheses)

    <ruby>
    日本 <rp>(</rp><rt>にほん</rt><rp>)</rp>
</ruby>
    日本 (にほん)

    Output: Provides parentheses around ruby text for browsers that don't support <ruby>.
