ROOT & DOCUMENT STRUCTURE ELEMENTS (9)

<!DOCTYPE>

Document Type Declaration

<!DOCTYPE html>


Output: Tells the browser this is HTML5 document.

<html>

HTML Root Element

<html lang="en">
    <!-- All content goes here -->
</html>

Output: Root container for entire HTML document.

<head>

Document Head

<head>
    <title>My Page</title>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="style.css">
</head>

Output: Contains metadata, title, links, styles.

<title>

Document Title

<title>My Awesome Website</title>

Output: Shows "My Awesome Website" in browser tab.

<body>

Document Body

<body>
    <h1>Hello World!</h1>
    <p>All visible content.</p>
</body>

Hello World!

All visible content.

<meta>

Metadata:

<meta charset="UTF-8">
       <meta name="viewport" content="width=device-width, initial-scale=1.0">
     <meta name="description" content="Free tutorials">
     <meta name="author" content="Code One">

Output: Provides metadata for browsers and search engines.

<link>

External Resource Link:

<link rel="stylesheet" href="styles.css">
<link rel="icon" href="favicon.ico">
<link rel="preconnect" href="https://fonts.googleapis.com">

Output: Links external CSS, icons, fonts.

<base>

Base URL:

<base href="https://www.example.com/" target="_blank">

Output: Sets base URL for all relative links.

<style>

Style Tag:

<style>
    .my-class { color: red; font-size: 20px; }
</style>
<p class="my-class">Styled text!</p>

Styled text!

Bold Text

Bold text
<strong>

Strong Text

<strong>Important text</strong>

Output: Displays bold text with strong importance.

Important text
<i>

Italic Text

<i>Italic text</i>

Output: Displays italicized text.

Italic text
<em>

Emphasized Text

<em>Emphasized text</em>

Output: Displays emphasized text (usually italic).

Emphasized text
<mark>

Highlighted Text

<p>This is <mark>highlighted</mark> text.</p>

Output: Highlights text with a yellow background.

This is highlighted text.

<small>

Small Text

<p>Normal <small>smaller text</small></p>

Output: Displays smaller text.

Normal smaller text

<del>

Deleted Text

<del>Old price $100</del>

Output: Displays deleted text (usually with a line through it).

Old price $100
<ins>

Inserted Text

<ins>New price $80</ins>

Output: Displays inserted text (usually with a underline).

New price $80
<sub>

Subscript

H<sub>2</sub>O

Output: Displays subscript text.

H2O
<sup>

Superscript

Output: Displays superscript text.

x2

Underlined Text

Output: Displays underlined text.

Underlined text

<blockquote>

Block Quote

<blockquote>
    The only limit is your imagination.
</blockquote>

Output: Displays a block-level quotation.

<q>

Inline Quote

<q>This is a short quote</q>

Output: Displays an inline quotation (usually with quotation marks).

This is a short quote
<cite>

Citation

<cite>The Mona Lisa</cite> by Leonardo da Vinci

Output: Displays a citation (usually in italics).

The Mona Lisa by Leonardo da Vinci
<bdo>

Bi-Directional Override

<bdo dir="rtl">This text goes right to left</bdo>

This text goes right to left
<bdi>

Bi-Directional Isolate

<ul>
    <li>User <bdi>John</bdi>: 80 points</li>
    <li>User <bdi>أحمد</bdi>: 90 points</li>
</ul>

Output: Isolates the directionality of text.

  • User John: 80 points
  • User أحمد: 90 points
<pre>

Preformatted Text

    function hello() {
        console.log("Hello!");
    }
</pre>

Output: Displays preformatted text.

    function hello() {
        console.log("Hello!");
    }
<code>

Code

<code>console.log("Hello World!");</code>

Output: Displays code.

console.log("Hello World!");

Keyboard Input

Press Ctrl + C to copy
<samp>

Sample Output

<samp>Hello World!</samp>

Output: Displays sample output.

Hello World!
<var>

Variable

<var>x</var> = <var>y</var> + <var>z</var>

Output: Displays a variable.

x = y + z
<span>

Inline Container

<span style="color: red;">RED text</span>

Output: Displays inline text with red color.

RED text
<div>

Block Container

<div style="background: lightblue; padding: 10px;">
    This is a div container
</div>

Output: Displays a block-level container.

This is a div container

LIST ELEMENTS

<ul> & <li>

Unordered List

<ul>
    <li>Apple</li>
    <li>Banana</li>
    <li>Orange</li>
</ul>

Output: Displays a bulleted list.

  • Apple
  • Banana
  • Orange
<ol> & <li>

Ordered List

<ol>
    <li>First step</li>
    <li>Second step</li>
    <li>Third step</li>
</ol>

Output: Displays a numbered list.

  1. First step
  2. Second step
  3. Third step

</dl>

Output: Displays a list of terms and descriptions.

HTML
HyperText Markup Language
CSS
